使い始める。

Clawdbotをあなたの環境に構築し、AIの力を解放するためのステップバイステップガイド。

セクション
導入ステップ 詳細設定 プロンプト例 コストとモデル

1. 導入ステップ

1

インストール

ClawdbotはNode.jsで動作します。まずはnpmを使用してグローバルにインストールします。

Terminal
npm install -g clawdbot
2

ゲートウェイの起動

サービス本体を起動します。これによりAIエージェントがアクティブになります。

Terminal
clawdbot gateway start
3

認証の設定

使用するAIモデル（Gemini, Claude等）のAPIキーを設定します。

Terminal
clawdbot config set model.apiKey 'YOUR_API_KEY'

2. 外部ツールとの連携

ClawdbotはDiscordやSlackをインターフェースとして使用できます。

Discord連携

Botトークンを取得し、以下のコマンドで設定を反映させます。即座にサーバーでClawdbotが応答を開始します。

clawdbot config set discord.enabled true
clawdbot config set discord.token '...'

3. 指示を出してみる

チャット欄で、まるで部下や同僚に頼むように指示を投げてください。

リサーチ
「今日のテック系ニュースを要約して」

Webを検索し、重要な情報をまとめて報告します。

開発
「このファイルをTypeScript化して」

ファイルを直接読み書きし、コードのリファクタリングを行います。
