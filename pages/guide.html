Clawdbot 導入・活用ガイド

詳細な導入ステップ

1
インストールと起動

ClawdbotはNode.js環境で動作します。端末（ターミナル）で以下のコマンドを実行して起動します。

# Clawdbotを起動する
clawdbot gateway start

# ステータスを確認する
clawdbot status
2
プロバイダーの設定

Discordなどのチャットツールと連携させるために、設定ファイル（config）を編集します。コマンドで直接設定することも可能です。

# Discordを有効にする例
clawdbot config set discord.enabled true
3
AIモデルの選択

使用するAI（GeminiやClaudeなど）を設定します。APIキーをセットすることで、Clawdbotが思考を開始します。

気になるコスト（料金）について

Clawdbot自体の利用はオープンソースのため無料ですが、以下の外部コストが発生する場合があります：

  • AI API利用料: AIモデル（Gemini, Claude, GPT等）の利用量に応じた従量課金。
  • サーバー電気代: 自宅サーバーで24時間稼働させる場合の電気代。
  • 外部サービス: 有料の外部ツール（Notionの特定プランなど）を連携させる場合。

※Geminiなどの無料枠を活用することで、コストを最小限に抑えることも可能です。

便利な指示の例

🔍 調査の依頼

「最新のAIニュースを3つピックアップして、専門用語を抜きにして要約して。」

📂 ファイル整理

「Downloadsフォルダにある昨日のPDFファイルを、プロジェクト名のフォルダに移動させて。」

🛠️ 開発サポート

「このHTMLファイルに、ダークモード切り替えボタンを追加するスクリプトを書いて。」

さらに使いこなすには

具体的な指示を出すほど、Clawdbotは正確に動きます。指示に迷ったら「何ができる？」と聞いてみてください。
